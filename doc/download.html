Download

Pictorial Consequences is written in Java, and so requires a Java Runtime Environment to run. It was written with version 1.2. To the best of our knowledge, it will work with the latest version as well.

Windows

Java Runtime Environment for Windows (5,329k)
You'll need to have this installed to run Pictorial Consequences. This copy is version 1.3.1.
Pictorial Consequences for Windows (176k)
Once you have the Java Runtime Environment installed you can download and run Pictorial Consequences on your computer.

UNIX

Modern UNIX systems have Java installed on them. We trust you to work out how to get it installed if this isn't the case. Just make sure you're running version 1.2 or greater. The archives below have compiled cross-platform versions (including source code). If you'd like to recompile them there is a GNU Makefile included for your convenience.
